‚è±Ô∏è Quick Start (5 Minutes)

  1. Get API Key: Visit console.groq.com
  2. Edit Configuration: Copy backend/.env.example to backend/.env and add your key
  3. Start Backend: Run start_backend.bat (Windows) or start_backend.sh (Mac/Linux)
  4. Start Frontend: Run start_frontend.bat (Windows) or start_frontend.sh (Mac/Linux)
  5. Open App: Go to http://localhost:3000
  6. Talk: Click the microphone button and speak!

üÜò Troubleshooting

Common Issues:

  • Backend won't start: Check Python version (3.9+), activate venv, reinstall packages
  • Microphone not working: Check browser/OS permissions, try different browser
  • API errors: Verify backend is running, check API key in .env
  • First response slow: Normal! Model loads on first request (2-3 sec)
